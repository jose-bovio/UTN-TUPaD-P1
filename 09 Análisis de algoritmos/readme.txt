TEMA ELEGIDO: ANALISIS DE ALGORITMOS

Elegimos este tema ya que queremos entender como todo el código que diseñamos, tipeamos y ejecutamos trabaja en el hardware, a nivel matemático, lógico, computacional
y como se traduce en gasto  de recursos. Para no solo hacer un código funcional sino también uno eficiente para nosotros o nuestros clientes.

Nos basaremos en el funcionamiento básico de la computadora, como procesa los cálculos aritméticos y en base a esto veremos cual es el mas difícil para la computadora y el mas fácil de procesar
En base a esto haremos un algoritmo para cada caso, que busque números pares en una n lista.

El objetivo será ver si realmente si usamos un algoritmo u otro, que a simple vista son caracteres, se termina traduciendo en un consumo real de recursos, produciendo grandes consumos de procesos,
energía eléctrica, ancho de banda, etc.
